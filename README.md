# 🏥 Hospital Management System (C++)

A console-based Hospital Management System developed in C++ that allows users to manage patients, doctors, appointments, and billing efficiently.

---

## 📌 Features

* ➕ Add new patients
* ➕ Add new doctors
* 📋 View all patients
* 📋 View all doctors
* 📅 Schedule appointments
* 🔍 Search patient by ID
* 🔍 Search doctor by ID
* 💳 Generate billing receipt
* 🧾 View all appointments

---

## 🛠️ Technologies Used

* C++
* Standard Template Library (STL)

  * `vector`
  * `algorithm`
  * `iomanip`
* Object-Oriented Programming (OOP)

## ⚙️ How to Run (Ubuntu/Linux)

### Step 1: Install g++

```bash
sudo apt update
sudo apt install g++
```

### Step 2: Compile the code

```bash
g++ main.cpp -o hospital
```

### Step 3: Run the program

```bash
./hospital
```
